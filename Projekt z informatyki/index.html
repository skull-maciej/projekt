HTML - historia powstania i standaryzacja

HTML - hyper text meta language, stworzony około roku 1989 obecnie standaryzowany przez konsorcjum W3C (http://www.w3c.org) Wykorzystywane standardy HTML:

  • HTML 3.x
  • HTML 4.x
  • HTML 5 - najnowoczesniejszy standard HTML, pozwalajacy na pisanie interaktywnych gier i aplikacji jedynie z wykorzystaniem kodu HTML i Javascipt.

Schemat budowy strony internetowej

Opis całej strony zawarty jest między znacznikami <html>,a </html>. Nagłówek strony wpisuje się pomiędzy <head>, a </head>, a w nim umieszcza się tytuł strony pośrodku <title>, a </title>. Tytuł pojawia się na pasku tytułowym okna przeglądarki. Zawartość strony umieszczamy wśród <body>, a </body>.

Podstawowa struktura strony:

  • <html>
    • <head>
      • <title>
        • tytuł strony
      • </title>
    • </head>
    • <body>
      • treść strony
    • </body>
  • </html>

Opis wybranych znaczników html

Strony WWW opisane za pomocą HTML, składają się z serii zagnieżdżonych znaczników. Ważne jest jednak, aby zamykając znaczik, zamknąć wszystkie znacziki otwarte wewnątrz niego, np.:

  • <p> <b> tworzenie strony internetowej jest fajne </p> </b> - źle
  • <p> <b> tworzenie strony internetowej jest fajne </b> </p> - dobrze

Tabele

Tabele tworzymy za pomocą znaczika <tabel>. Poszczególne wiersze tabeli porzedza się znacznikiem <tr>, a komórki znaczikiem <td>. Znaczniki te występują w parze ze znaczikiem zamykającymi

Obrazki

Aby wstawić obraz na stronę, należy zastosować znaczik <img>, zwykle z atrybutem src. Ogólna postać znacznika jest następująca: <img src="nazwa.gif">, nazwa.gif określa nazwę pliku z grafiką Strony HTML obsluguja nastepujace formaty graficzne: gif, jpg, png.

Align

Atrybut definiuje wyrównanie względem innych elementów w poziomie lub w pionie. Różne elementy HTML korzystają z różnych wartości atrybutu, więc warto zwracać uwagę na miejsce jego użycia.
Przykłady zastoswania align:

<p align=right>tekst</p>

<p align=center>tekst</p>

<p align=left>tekst</p>

Kolor czcionki oraz tła

Można zmieniać kolor czcionki oraz kolor tła.

Zmienianie koloru czcionki:

  • <font color=kolor> tekst </font>

Zmienianie koloru tła:

  • <body bgcolor=kolor>

Uwaga: W najnowszym standarcie opisu języka HTML, tzw. HTML5, znacznik font jest nie zalecany, dlatego obecnie bardziej prawidlowym podejsciem jest korzystanie z tzw. formularzy stylów CSS.

W miejscu "tekst" wpisujesz tekst, który chcesz zmienić kolor, a w "kolor", wybrany kolor przez ciebie, lecz pamiętaj,
że nazwy kolorów muszą być wpisane w języku angielskim i nie wszystkie kolory da się wyświetlić.

Hyperłącza

Ważne też są linki, dzięki którym możesz np. nawigować stronę. Możesz je wstawiać za pomocą tego tagu:

  • <a href=link>Kliknij tutaj</a>
  • Kliknij tutaj

W miejscu ''link'' po prostu wklejasz adres strony, na którą ma po kliknięciu przenosić. Zamiast ''kliknij tutaj'' możesz wstawić dowolny tekst. Wstawiając zamiast tego tekstu kod obrazka, będziemy mogli przejść na inną stronę klikając na obrazek.

  • <a href=link><img src=rysunek.gif></a>

W miejscu "rysunek.gif" wklejasz nazwę pliku, a w "link" adres strony.

Arkusze stylu - CSS

To język służący do opisu formy prezentacji (wyświetlania) stron WWW. Arkusz stylów składa się z reguł określających styl dla wybranych elementów dokumentu. Reguła składa się z selektora oraz deklaracji. Selektor określa grupę elementów, którego ma dotyczyć deklaracja. Deklaracja określa formatowanie i składa się z nazwy jednej z właściwości i jej wartości napisanej po dwukropku. Deklaracja musi być otoczona nawiasami klamrowymi.

Wzór css:

selektor { właściwość: wartość } np.

h1 {color:red} - to oznacza, że wszyskie nagłówki pierwszego poziomu będą koloru czerwonego.

Dane do arkuszy stylu muszą się znaleść pomiędzy odpowiednimi znaczikami:<style type="text/css">, a </style>.

Javascipt - dodanie życia do strony HTML

Aby dodac troche dynamiki do strony HTML, mozna wykorzystac jezyk programowania Javascipt, ktory jest interpretowany przez wszystkie główne przegladarki HTML (WWW). Kod programu (funkcji) Javascipt musi znajdować sie pomiędzy specjalnymi tagami <script> i </script>.

Kod funkcji do pomniejszania/powiekszania obrazka:

				<script language="javascript">
				function resize(w,h){
					// wyszukaj z dokumentu HTML element o identyfikatorze 
					// rownym 'horse' (atrybut id='horse)
					var e = document.getElementById('horse');
					//zmien styl tego elementu, ustaw zadana szerokosc
					e.style.width = w+'px';
					//zmien styl tego elementu, ustaw zadaną wysokość
					e.style.height = h+'px';
				}
				</script>
